Lab::VISA

Lab::VISA allows to perform test and measurement tasks with Perl scripts. It provides an interface to National Instruments' NI-VISA library, making the standard VISA calls available from within Perl programs. Dedicated instrument driver classes relieve the user from taking care for internal details and make data aquisition as easy as

$voltage = $multimeter->read_voltage();

The Lab::VISA software stack comprises three parts that are built on top of each other and provide increasing comfort. Your measurement scripts can be based on any of these stages.

The lowest level is Lab::VISA. It makes the NI-VISA library accessible from Perl and thus allows to make any standard VISA call.

The modules in the Lab::Instrument package make communicating with instruments easier by silently handling the protocol overhead involved.

Package Lab::Tools is the highest abstraction layer. These modules provide support for writing good measurement scripts. They offer means of saving data and related meta information to disk, plotting data etc.

These packages together are referred to as the Lab::VISA system. Encapsulating the complexity of VISA calls into a straightforward to use library, the Lab::VISA system is designed to make you happy.

How to obtain

The packages are free software and can be downloaded from CPAN. Follow these links for

  • Lab::VISA,
  • Lab::Instrument, and
  • Lab::Tools.

Documentation

Quite some documentation is available online in HTML and PDF format. This documentation contains a tutorial on using Lab::VISA and a description of the various APIs.

Additionally, a powerpoint presentation is contained in the Lab::VISA package, and is available online as PDF. The examples shown in the presentation are contained in the package as well.

There is a mailing list set up for Lab::VISA (lab-visa-users mailing list). This mailing list is the right place to give feedback and ask for help.

Also useful is the NI reference on VISA resource names, VISA error codes, the NI-VISA User Manual, and the NI-VISA Programmer Reference Manual.

Status

Although this software has been used for years in real world measurements by its developers, it remains work in progress. Please bear with us while we constantly improve code and documentation.

Lab::VISA is currently developed and employed at nanophysics group, LMU München and mesoscopic physics group, Uni Regensburg. Users have reported further applications in academic and industrial r&d environments.
