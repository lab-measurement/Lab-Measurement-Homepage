Lab::VISA

Lab::VISA allows to perform test and measurement tasks with Perl scripts. It provides an interface to National Instruments' NI-VISA library, making the standard VISA calls available from within Perl programs. Dedicated instrument driver classes relieve the user from taking care for internal details and make measurements as easy as

$voltage = $multimeter->read_voltage();

The Lab::VISA software stack comprises three parts that are built on top of each other and provide increasing comfort. Your measurement scripts can be based on each of these stages.

The lowest level is Lab::VISA. It makes the NI-VISA library accessible from Perl and thus allows to make any standard VISA call.

The modules in the Lab::Instrument package make communication with instruments easier by silently handling the protocol involved.

Package Lab::Tools is the highest abstraction layer. These modules provide support for writing good measurement scripts. They offer means of saving data and related meta information to disk, plotting data etc.

These packages are together referred to as the Lab::VISA system. Encapsulating the complexity of VISA calls into a straightforward to use library, it is designed to make any lab power user happy.

How to obtain

The packages are free software and can be downloaded from CPAN. Follow these links for

  • Lab::VISA,
  • Lab::Instrument, and
  • Lab::Tools.

Documentation

Quite some documentation is available online in HTML and PDF format. This documentation starts with a general introduction into using Lab::VISA and continues with a description of the various APIs.

Additionally, a powerpoint presentation is contained in the Lab::VISA package, and is available online as PDF. The examples discussed in the presentation can be found in the 'Tutorial' directory of Lab::VISA. These examples were run in realtime during the original talk.

There is a mailing list set up for Lab::VISA (lab-visa-users mailing list). This mailing list is also the right place to give feedback and ask for help.

Status

Although this software has been used for years in real world measurements by its developers, it remains work in progress. Please bear with us while we constantly improve code and documentation.

Lab::VISA is currently developed at nanophysics group, LMU München and mesoscopic physics group, Uni Regensburg. Users have reported further uses in academic and industrial r&d environments.

Authors

Lab::VISA and friends were originally developed by Daniel Schröer and are currently continued by Andreas K. Hüttel and Daniel Schröer. Most of the documentation was written by Daniel Schröer.
